# 자동 높이/너비 검증 체크리스트

기준: 자동 높이/너비 상자는 편집 화면과 미리보기 화면 모두에서 `max(명시 최소 높이/너비, 내용+여백 필요 크기)`로 계산한다.

대상 템플릿: `0c6197e9-a221-46a9-9367-1e6bc84938c4`

최종 브라우저 확인: `http://localhost:3001/templates/edit?templateId=0c6197e9-a221-46a9-9367-1e6bc84938c4`에서 직접 실행.

핵심 판정:

- 저장된 `data-template-frame-auto-height-base`/`width-base`가 있어도 `*-base-explicit="true"`가 아니면 현재 계산 높이/너비로 보고 최소값으로 쓰지 않는다.
- 편집 화면과 미리보기 화면은 같은 계산 기준을 사용한다.
- 미리보기에서 value 상자의 예시 텍스트가 비워져도 명시 최소값과 빈 내용+여백 기준으로 다시 계산한다.
- normalized band 기반 상자는 자동 크기 변경 뒤에도 같은 행/열 경계를 공유하며, 인접 상자끼리 겹치면 실패로 본다.

| 상태 | 실행 | 기대 결과 | 확인 결과 |
| --- | --- | --- | --- |
| 편집 | value 상자 텍스트 삭제 | 명시 최소값이 없으면 내용+여백 크기까지 줄어든다 | 통과: `band-11-cell-2` 83px -> 40px |
| 편집 | value 상자 텍스트 입력 | 내용+여백 필요 크기까지 늘어난다 | 통과: `band-11-cell-2` 40px -> 83px |
| 편집 | value 상자 텍스트 삭제/복구 후 주변 행 | 주변 normalized band 행/열 경계가 유지되고 상자가 겹치지 않는다 | 통과: `band-4`~`band-8` 겹침 0건 |
| 편집 | key/독립 상자 텍스트 유지 | 값이 있는 텍스트는 잘리지 않는다 | 통과: key 텍스트 기준 높이 유지 |
| 편집 | 상하/좌우 여백 변경 | 변경된 여백이 자동 크기 계산에 반영된다 | 통과: `band-11-cell-2` 40px -> 55px |
| 편집 | 상자 역할 key/value/independent 변경 | value는 미리보기에서 빈 값 기준, key/independent는 텍스트 유지 기준으로 계산된다 | 통과: role을 key로 바꾸면 미리보기 텍스트 유지 |
| 편집 | 최소 높이/너비 입력 | 명시 최소값보다 작아지지 않는다 | 통과: `band-14-cell-2` 최소 50px 적용 |
| 미리보기 | value 상자 빈 값 | 명시 최소값이 없으면 빈 내용+여백 크기까지 줄어든다 | 통과: `band-18-cell-2` 73px -> 25px, `band-11-cell-2` 88px -> 40px |
| 미리보기 | value 상자 값 입력 | 입력한 값+여백 필요 크기까지 늘어난다 | 통과: `band-14-cell-2` 50px -> 69px |
| 미리보기 | value 상자 값 삭제 | 다시 명시 최소값 또는 빈 내용+여백 크기까지 줄어든다 | 통과: `band-14-cell-2` 69px -> 50px |
| 미리보기 | value 상자 입력/삭제 후 주변 행 | 주변 normalized band 행/열 경계가 유지되고 상자가 겹치지 않는다 | 통과: `band-4`~`band-8` 겹침 0건 |
| 미리보기 | 자동 너비 value 상자 빈 값 | 명시 최소 너비가 있으면 그 너비를 하한으로 유지한다 | 통과: `band-0-header` 300px |
| 미리보기 | 자동 너비 value 상자 값 입력 | 입력한 값+여백 필요 너비까지 늘어난다 | 통과: `band-0-header` 300px -> 1094.1px |
| 미리보기 | `signature_image` 런타임 상자 | 캔버스 해상도는 레이아웃 내용으로 쓰지 않되, `지우기` 버튼처럼 실제 보이는 내용은 자동 높이 계산에 반영한다 | 통과: `band-8-cell-4` 버튼이 상자 안에 포함되고 겹침 0건 |
| 공통 | peer edge 연결 상자 | 연결된 상자들은 edge 관계를 유지한다 | 통과: 편집/미리보기/편집 복귀 모두 `status-history-1:left` = `band-3-cell-2:right` = `band-4-cell-2:right`, 겹침 0건 |
| 공통 | 간격 설정 그룹 | 자동 크기 후 설정된 간격을 유지한다 | 통과: 자동 크기 후 preserved spacing pass 실행 및 시각 붕괴 없음 |
